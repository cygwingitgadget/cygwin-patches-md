**Thread**: [thread](18-58-00-1.thread.md#:~:text=2019/04/20/18-58-00-4)

Don't set lct_termination_evt to NULL too early in
fhandler_fifo::stop_listen_client.  Doing so leads to an "Invalid
Handle" error in WFMO.
