**Thread**: [thread](08-57-34.thread.md#:~:text=2021/03/04/08-57-34)

- Restoring console mode fails in the following scenario.
```
   1) Start cygwin shell in command prompt.
   2) Run 'exec chcp.com'.
  This patch fixes the issue.
```
