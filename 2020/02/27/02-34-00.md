**Thread**: [thread](02-34-00.thread.md#:~:text=2020/02/27/02-34-00)

- This patch makes some detailed behaviour of ESC sequences such as
```
  "CSI Ps L" (IL), "CSI Ps M" (DL) and "ESC M" (RI) in xterm mode
  match with real xterm.
```
