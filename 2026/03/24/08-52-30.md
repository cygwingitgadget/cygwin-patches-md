Dear Team,

We are migrating from Bitbucket to GitHub as part of our platform modernization initiative.
Action is required on your personal repositories (under ~username namespaces) no later than March 31, 2026.
--------------------------------------------------------------------------------------------------------------------------
🗓️ Key Dates

  *   Jan 31, 2026 – New personal repo creation disabled
  *   Mar 31, 2026 – Final migration/cleanup deadline
  *   Apr 1, 2026 – Unmigrated repos Archived
--------------------------------------------------------------------------------------------------------------------------
📋 What You Need to Do

  *   Review your personal repositories in Bitbucket
  *   Inactive personal repositories → We will not migrate inactive repositories; they will be archived
  *   Active personal repositories →
     *   Personal repositories will never be migrated by this program.
     *   If an owner wants a personal repository migrated, they must first convert it to a project-owned repository and then raise a JIRA migration request<https://jira.analog.com/servicedesk/customer/portal/15/create/1403>
---------------------------------------------------------------------------------------------------------------------------
🎫 JIRA Request Must Include

  *   Repo name & Bitbucket URL
  *   Target GitHub org
  *   Repo description

⚠️ Repos not migrated by March 31 will be archived. Recovery will require JIRA approval with justification.
